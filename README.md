# android-sqlserver-employee-lookup
Aplicación Android desarrollada en Java que permite consultar información de empleados mediante conexión directa a una base de datos SQL Server y mostrar los resultados en tiempo real.

---Sistema de Consulta de Personal para Android---

---Descripción---

Desarrollé una aplicación móvil para Android enfocada en la consulta de información de personal almacenada en una base de datos SQL Server. El sistema permite realizar búsquedas utilizando el número de identificación del empleado y visualizar información relevante como nombre, dirección, teléfono y salario desde una interfaz móvil intuitiva.

El proyecto integra tecnologías de desarrollo móvil y bases de datos empresariales, permitiendo establecer comunicación entre una aplicación Android y un servidor SQL Server para recuperar información en tiempo real.

---Funcionalidades---

* Consulta de empleados mediante número de identificación (DNI).
* Conexión a base de datos SQL Server.
* Recuperación de información en tiempo real.
* Visualización de datos personales del empleado.
* Interfaz gráfica desarrollada para dispositivos Android.
* Comunicación cliente-servidor.
* Manejo básico de errores de conexión y consultas.

---Tecnologías Utilizadas---

---Desarrollo Móvil---

* Java
* Android Studio
* Android SDK
* XML para diseño de interfaces

---Base de Datos---

* Microsoft SQL Server
* JDBC (JTDS Driver)
* Consultas SQL

### Comunicación

* Conexión mediante red TCP/IP
* Comunicación entre aplicación móvil y servidor de base de datos

---Habilidades Demostradas---

* Desarrollo de aplicaciones Android nativas.
* Diseño de interfaces móviles.
* Integración de bases de datos en aplicaciones móviles.
* Conexión remota a SQL Server.
* Ejecución de consultas SQL.
* Manipulación de resultados mediante ResultSet.
* Manejo de eventos e interacción con el usuario.
* Arquitectura cliente-servidor.
* Gestión básica de errores y excepciones.

---Funcionamiento del Sistema---

1. El usuario ingresa un número de identificación (DNI).
2. La aplicación establece conexión con la base de datos SQL Server.
3. Se ejecuta una consulta SQL para localizar el registro correspondiente.
4. Si el empleado existe, se recuperan sus datos.
5. La información se muestra automáticamente en la interfaz móvil.
6. En caso de error o registro inexistente, se notifica al usuario.

El objetivo de este proyecto fue fortalecer mis conocimientos en desarrollo móvil Android y conectividad con bases de datos empresariales, implementando una solución capaz de consultar información almacenada en SQL Server desde una aplicación móvil desarrollada en Java.
